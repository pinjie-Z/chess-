## 📋 运行前准备

### 环境要求

| 软件 | 版本 | 下载链接 |
|------|------|----------|
| JDK | 17 | [https://adoptium.net/temurin/releases/?version=17](https://adoptium.net/temurin/releases/?version=17) |
| Maven | 3.6+ | [https://maven.apache.org/download.cgi](https://maven.apache.org/download.cgi) |

### 环境变量配置

**Windows：**

1. 设置 `JAVA_HOME`：指向 JDK 17 安装目录
   ```powershell
   setx JAVA_HOME "C:\Program Files\Eclipse Adoptium\jdk-17.0.x.x"
   ```

2. 设置 `MAVEN_HOME`：指向 Maven 安装目录
   ```powershell
   setx MAVEN_HOME "D:\apache-maven-3.9.x"
   ```

3. 将以下路径添加到系统 `PATH`：
   - `%JAVA_HOME%\bin`
   - `%MAVEN_HOME%\bin`

### 验证环境

打开新的命令行窗口，执行以下命令验证：

```powershell
java -version
mvn -v
```

输出应显示 Java 17 和 Maven 3.6+ 的版本信息。

## 🚀 完整运行命令

### 1. 进入项目目录
```powershell
cd chess--main\chess
```

### 2. 编译打包
```powershell
mvn clean package -DskipTests
```

### 3. 启动服务器
```powershell
mvn spring-boot:run
```

### 4. 访问游戏
打开两个浏览器标签页，访问：
```
http://localhost:8887
```

---

### ⚠️ 如果端口被占用（提示 "port 8887 already in use"）

先查找并杀死占用进程：
```powershell
netstat -ano | findstr :8887
```
找到 PID 后：
```powershell
taskkill /F /PID <你的PID>
```
